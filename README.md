# RCCG Rehoboth Chapel — Church Website

A responsive church web platform architected with Next.js and Tailwind CSS, featuring a real-time Sunday service countdown, dynamic hero slideshow, and a custom navy/wine/gold design system.

## Features

- **Live Service Countdown** — Dynamically calculates and displays the time remaining until the next Sunday service, engineered with Nigeria (WAT) timezone logic.
- **Ongoing Service Indicator** — Automatically switches to a "Live Now" state during active service hours (9:00 AM – 11:30 AM WAT).
- **Auto-Rotating Hero Slideshow** — Crossfading background image carousel for an immersive landing experience.
- **Responsive Navigation** — Sticky navbar with active-link highlighting, smooth hover transitions, and a fully animated mobile menu.
- **Custom Design System** — Wine-to-purple gradient theme accented with gold, applied consistently across navigation and interactive elements.
- **Optimized Performance** — Built on Next.js App Router with image optimization via `next/image`.

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

Clone the repository and install dependencies:

```bash
git clone <your-repo-url>
cd <your-project-folder>
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
